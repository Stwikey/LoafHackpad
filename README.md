- 6x Cherry MX Switches
- 2x SK6812 MINI Leds
- 1x XIAO RP2040
- 6x Blank DSA keycaps
- 4x M3x16mm Screws
- 4x M3x5mx4mm heatset inserts

- KMK Firmware
- 3D Printed Case (TOP.stl, BOTTOM.stl)
